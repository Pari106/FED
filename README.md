<details>
  <summary>uitwerken voor 2<sup>e</sup> voortgang</summary>

  ### Stand van zaken
Ik ben al heel erg vooruit gekomen met mijn code, er staat voor mijn gevoel al een goede basis. Het begint al op de echte Melkweg website te lijken.
